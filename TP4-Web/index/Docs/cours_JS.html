LA STRUCTURE DE BASE

Lorsque le code JavaScript se trouve dans un fichier séparé (.js), la structure est simple[cite: 113]:

/* zone d'en-tête */
/* déclaration des variables globales */
/* définition des fonctions */
/* CORPS DU PROGRAMME */

LES COMMENTAIRES

Les commentaires permettent une relecture de programme pour le programmeur lui-même[cite: 155].

  • // : Commentaire sur une seule ligne[cite: 160].
  • /* ... */ : Commentaire multi-lignes[cite: 161].

LES DECLARATION DE VARIABLES

JavaScript est dit "faiblement typé"[cite: 185]. Une variable est déclarée avec var ou let[cite: 191].

Les types primitifs :

Number, String, Boolean, Null, Undefined, BigInt, Object, et Symbole[cite: 193].
