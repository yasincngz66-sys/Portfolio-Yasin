Langage JavaScript : Les bases

1. Structure de base

Le JavaScript peut être intégré directement dans le HTML avec la balise <script>.


    <script src="code.js"></script>

2. Les Variables

JavaScript est un langage faiblement typé.

On déclare une variable avec let ou var.


    let a = 10; // Déclaration d'un Number [cite: 291, 649]
    const pi = 3.14; // Déclaration d'une constante [cite: 281, 639]

3. Les Commentaires

  • // pour un commentaire sur une ligne
  • /* ... */ pour plusieurs lignes

4. Fonctions d'entrée/sortie

  • window.alert("Message"); : Affiche une boîte d'alerte.
  • window.confirm("Continuer ?"); : Demande une confirmation.
  • window.prompt("Votre nom ?"); : Demande une saisie à l'utilisateur.
