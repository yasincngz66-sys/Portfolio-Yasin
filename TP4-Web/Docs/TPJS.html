TP8 : LES TABLEAUX

INTRODUCTION

Dans ce TP, il est demandé d'écrire des fonctions dites utilitaires permettant de travailler avec les tableaux. Ces fonctions seront ensuite utilisables dans d'autres TP[cite: 4].

Le tableau utilisé contient N éléments. Interdit dans cet exercice de pouvoir ajouter, d'insérer ou supprimer une case de ce dernier[cite: 25]. Ce tableau est déclaré de la façon suivante :

let N = parseInt(prompt("Combien de case pour le tableau ?"));
let A = new Array(N);
let nbElemRempli = 0;

Méthodologie pour chaque fonction :

  1. Boite noire : réfléchir aux entrées et aux sorties.
  2. Dans le fichier .js coder la fonction.
  3. Ajouter le mode d'emploi de chacune des fonctions.
  4. Tester la fonction (avec l'inspecteur d'élément directement).

EXERCICE 1

Écrire une fonction initialiserTab1() permettant d'initialiser les N cases d'un tableau avec la valeur 0.

EXERCICE 2

Écrire une fonction initialiserTab2() permettant d'initialiser les N cases d'un tableau avec une valeur passée en paramètre.
